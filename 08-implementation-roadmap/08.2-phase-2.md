## 6. 验收演示场景

**场景名称：** 文本摘要工作流

```
[Input Node]        用户输入一段长文本
     ↓
[LLM Call Node]     GPT-4 执行摘要 (prompt: "请用 3 句话概括以下内容:\n{input}")
     ↓
[Output Node]       输出摘要结果
```

**执行流程：**
1. 在画布上放置 3 个节点并连线
2. 配置 LLM 节点参数 (model, prompt, temperature)
3. 点击验证 → 显示 "验证通过"
4. 点击 Run → 节点颜色依次变蓝 → 变绿
5. Output 节点显示摘要结果
